## 🎨 Design Guidelines

### Color Palette:
- Primary: `#0f172a` (slate-900)
- Secondary: `#1e293b` (slate-800)
- Accent: `#64748b` (slate-500)
- Background: `#fafafa`
- Text: `#0f172a` / `#ffffff`

### Typography:
- Heading: Space Grotesk (Bold/Semibold)
- Body: Plus Jakarta Sans (Regular/Medium)

### Animation Style:
- Smooth transitions (300-500ms ease-out)
- Subtle scale effects (1.02x on hover)
- Fade in/out dengan slide
- No jarring movements

### UI Elements to Show:
✅ Clean navigation bar
✅ Bottom tab bar (mobile)
✅ Post cards dengan rounded corners
✅ Avatar circles
✅ Like/comment buttons
✅ Story circles
✅ Chat bubbles
✅ Notification badge
✅ Video player controls

### What NOT to Show:
❌ Ads atau sponsored content
❌ Cluttered UI
❌ Too many notifications
❌ Confusing navigation
❌ Loading spinners (keep it smooth)

## 🎥 Production Tips

### Recording:
1. Use screen recording tool (OBS Studio / QuickTime)
2. Record at 60fps for smooth playback
3. Use cursor highlight untuk interaction demo
4. Record multiple takes untuk setiap scene

### Editing:
1. Use Adobe Premiere / Final Cut / DaVinci Resolve
2. Add motion graphics untuk text overlay
3. Color grade untuk consistency
4. Add subtle sound effects
5. Export: H.264, 1080p, 30fps, High quality

### Music:
- Royalty-free dari Epidemic Sound / Artlist
- Genre: Lo-fi, Ambient, Modern Electronic
- Mood: Calm, Professional, Modern
- BPM: 80-100 (not too fast)

### Text Animation:
- Fade in from bottom (20px offset)
- Duration: 0.3s ease-out
- Stay on screen: 2-3 seconds
- Fade out: 0.2s
